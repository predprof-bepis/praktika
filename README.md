# Проект Кейс 3
Программа для работы с заявками абитуриентов. Имеет версию для компьютеров и версию в виде сайта. Умеет генерировать PDF-отчёты, вычислять проходной балл и т.д.
# Как развернуть?
 - Шаг 1. Клонируйте репозиторий
 - Шаг 2. Установите зависимости
   `python -m pip install -r requirements.txt`
 - Шаг 3. Пропишите в терминал `python web\app.py`.
